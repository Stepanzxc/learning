# learning

## Инициалиазация прокта через git 
```shell
git init
```

## Посмореть статуст измененией 
```shell
git status
```

## Добавить файлы в git 
```shell
git add <filename>
```

## Создать коммит 

```shell
git commit -m <name commit>
```
## Создание main ветки 
```shell
git branch -M main
```
