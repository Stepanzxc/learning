# learning

## Инициалиазация прокта через git 
```shell
git init
```

## Посмореть статуст измененией 
```shell
git status
```

## Добавить файлы в git 
```shell
git add <filename>
```

## Создать коммит 

```shell
git commit -m <name commit>
```
## Создание main ветки 
```shell
git branch -M main
```

## push в ветку  
```shell
git push origin main
```
## создать ветку  
```shell
git checkout -b "название ветки"
```
## просмотр веток 
```shell
git branch
```
